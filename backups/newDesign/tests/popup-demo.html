🎯 Demo Components

This demo page showcases two shared components from the AI Studio design system:

📧 Contact Popup Component

  • Smooth fade-in/out animations with backdrop blur
  • Form validation for all required fields
  • Loading states with animated spinner
  • Success and error message displays
  • EmailJS integration for real email sending
  • WhatsApp fallback when email fails
  • Responsive design for all screen sizes
  • Keyboard navigation (ESC to close)
  • Auto-focus on first input field

🧭 Shared Menu Component

  • Fixed navigation bar with backdrop blur
  • Responsive mobile hamburger menu
  • Multi-level dropdown navigation
  • Active page detection and highlighting
  • Smooth hover animations and transitions
  • Shopping cart integration
  • Contact popup integration for "Sign Up" buttons
  • Consistent styling across all pages

✅ Both Components Active: The menu is automatically injected at the top, and the popup opens when you click "Sign Up Today" in the menu!

Press ESC to close the popup, or click outside the modal area.
